## 快速开始

### 登录系统

1. 访问系统地址：`http://your-domain.com`
2. 输入用户名和密码
3. 点击"登录"按钮

### 界面导航

系统主界面包含以下菜单：

- **报表设计**：创建和编辑传统报表
- **报表预览**：查看已发布的报表
- **大屏设计**：创建数据可视化大屏
- **图表编辑**：单独编辑图表组件
- **数据源管理**：配置数据库连接

## 核心功能

### 报表设计

报表设计器提供了以下功能：

- **画布编辑**：拖拽单元格进行布局
- **数据绑定**：绑定数据源和字段
- **样式设置**：设置单元格样式、合并单元格
- **公式编辑**：使用表达式计算数据
- **预览发布**：实时预览和发布报表

### 大屏设计

大屏设计器用于创建可视化数据大屏：

- **组件库**：拖拽图表、文本、图片等组件到画布
- **属性面板**：配置组件的样式、数据、交互
- **图层管理**：管理组件的层级、可见性、锁定
- **实时预览**：随时预览大屏效果
- **全屏展示**：支持全屏展示模式

### 图表编辑

图表编辑器提供了丰富的图表类型：

**基础图表**
- 柱状图
- 折线图
- 面积图
- 散点图

**饼图**
- 基础饼图
- 环形图
- 南丁格尔玫瑰图

**关系图**
- 树状图
- 关系图
- 力导向图

**地理图表**
- 地图散点
- 地图热力
- 航线图

## 报表管理

### 创建报表

1. 进入"报表设计"页面
2. 点击"新建报表"按钮
3. 选择数据源和数据表
4. 拖拽字段到画布
5. 配置样式和公式
6. 点击"保存"按钮

### 编辑报表

1. 在报表列表中找到要编辑的报表
2. 点击"编辑"按钮
3. 修改报表配置
4. 点击"保存"按钮

### 发布报表

1. 编辑报表后，点击"发布"按钮
2. 设置报表参数（如需要）
3. 点击"确认发布"

### 预览报表

1. 进入"报表预览"页面
2. 选择要预览的报表
3. 输入参数（如需要）
4. 点击"查询"按钮

## 大屏设计

### 创建大屏

1. 进入"大屏设计"页面
2. 点击"新建大屏"按钮
3. 输入大屏名称和描述
4. 设置画布尺寸（宽高）
5. 点击"确认"

### 添加组件

1. 从左侧"组件库"中选择组件类型
2. 拖拽组件到画布
3. 调整组件位置和大小

### 配置数据

1. 选中画布上的组件
2. 在右侧"属性面板"中选择"数据"标签
3. 选择数据源和数据表
4. 配置字段映射和过滤条件

### 样式设置

1. 选中组件
2. 在"属性面板"中选择"样式"标签
3. 设置颜色、字体、边框等样式
4. 实时查看效果

### 预览和发布

1. 点击右上角"预览"按钮
2. 全屏查看大屏效果
3. 点击"关闭"返回编辑
4. 确认无误后点击"保存"

## 图表编辑

### 选择图表类型

1. 进入"图表编辑"页面
2. 在左侧"图表类型"面板中选择图表分类
3. 查看图表描述和示例
4. 点击选择图表类型

### 配置数据

1. 在"数据配置"面板中选择数据源
2. 选择数据表
3. 配置维度和指标
4. 设置过滤条件

### 设置样式

1. 在"属性面板"中选择"样式"标签
2. 设置图表主题
3. 配置颜色、标题、图例
4. 调整坐标轴和提示框

### 实时预览

1. 修改配置后，右侧预览区自动更新
2. 点击"刷新"按钮强制刷新
3. 查看数据加载状态

## 数据源配置

### 添加数据源

1. 进入"数据源管理"页面
2. 点击"添加数据源"按钮
3. 选择数据库类型
4. 填写连接信息：
   - 数据源名称
   - 主机地址
   - 端口号
   - 数据库名称
   - 用户名和密码
5. 点击"测试连接"
6. 连接成功后点击"保存"

### 测试连接

1. 在数据源列表中找到目标数据源
2. 点击"测试"按钮
3. 查看测试结果
4. 根据错误提示调整配置

### 查看元数据

1. 点击数据源的"表列表"链接
2. 查看所有数据表
3. 点击表名查看字段列表
4. 查看字段类型和注释

## 常见问题

### 登录失败

**问题**：输入正确的用户名密码后无法登录

**解决方案**：
1. 检查密码大小写
2. 清除浏览器缓存
3. 联系管理员重置密码

### 数据源连接失败

**问题**：测试数据源连接时失败

**解决方案**：
1. 检查数据库服务是否运行
2. 验证主机地址和端口
3. 确认用户名密码正确
4. 检查防火墙设置

### 报表加载慢

**问题**：报表加载时间过长

**解决方案**：
1. 检查数据源连接速度
2. 优化 SQL 查询
3. 减少数据量
4. 使用缓存机制

### 图表不显示

**问题**：图表区域显示空白

**解决方案**：
1. 检查数据配置是否正确
2. 确认数据源连接正常
3. 查看浏览器控制台错误
4. 验证字段映射

### 大屏响应慢

**问题**：大屏交互卡顿

**解决方案**：
1. 减少组件数量
2. 简化图表配置
3. 优化数据查询
4. 使用数据缓存
